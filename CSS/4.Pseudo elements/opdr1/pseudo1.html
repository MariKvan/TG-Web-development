Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

    ::first-line

  1. Het :: first-line pseudo-element wordt gebruikt om een speciale stijl toe te voegen aan de eerste regel van een text.

    voorbeeld met p element:

    You can use the ::first-line pseudo-element to add a special effect to the first line of a text. Some more text. And even more, and more, and more, and more, and more, and more, and more, and more, and more, and more, and more, and more.

    2. ::first-letter

  2. Het :: first-letter pseudo-element wordt gebruikt om een speciale stijl toe te voegen aan de eerste letter van een text

    3. voorbeeld met h5 element

    You can use the ::first-letter pseudo-element to add a special effect to the first character of a text!

  3. ::before

    Het :: before pseudo-element kan worden gebruikt om wat inhoud in te voegen vóór de inhoud van een element.

  4. ::after

    Het :: after pseudo-element kan worden gebruikt om wat inhoud in te voegen na de inhoud van een element.
