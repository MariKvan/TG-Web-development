Opdracht 1 Je kunt op 3 verschillende manieren CSS in je HTML inladen.Op welke 3 manieren kun je dat doen?
Leg ook voor elke wijze uit wat het precies inhoud.

  • Intern - door een <style> element in de <head>sectie te gebruiken
  • Extern - door een <link> element te gebruiken om naar een extern CSS-bestand te linken
  • Inline - door het style attribuut in HTML-elementen te gebruiken
