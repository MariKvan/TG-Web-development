Er zijn verschillende selectors, bijvoorbeeld;

  • type selectors
  • descendant selectors
  • class selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

antwoord

    type selectors

  • The CSS type selector komt overeen met elementen door node name. met andere woorden, het selecteerd alle elementen van de gegeven type binnen een document.
    voorbeeld

    • descendant selectors

  • De descendant selector is elke selector met witruimte tussen twee selectors {} zonder combinator.
    Neem bijvoorbeeld ul li {}. Het betekent 'elk list item dat afstamt van een ongeordende lijst'.
    Descendant betekent overal genest in de DOM-structuur.
    voorbeeld

    • class selectors

  • De CSS class selectorkomt overeen met elementen op basis van de inhoud van hun class attribuut.
    Het is een uitstekende manier om de HTML-elementen op te maken door ze te groeperen op basis van een aantal specifieke attributen
    voorbeeld
    *dus alle li.spacious elementen krijgen een margin van 2em
